# PhotoView
An application to put up some photos on the desktop.

1.  
Dragging the middle can adjust the window position.  
Press ESC to close the specified single image.  
Dragging the right or bottom edge can adjust the window size.  
The mouse wheel can adjust the image size.  
Ctrl + mouse wheel can quickly adjust the image size.  
Space + drag the middle to adjust the image position.  
Tab resets the image position.

2.  
Added screenshot functionality, single image pinning, switching to grayscale image, and window locking.  
Optimized the issue where the image size might reset when moving the image and the issue where the reference image wouldn't close when the window is closed.  
Press C to view the grayscale image, press L to lock the window, and right-click the image to bring up the menu.

3.  
Open clipboard image.  
Pick up color.  
Drag to open an image.

4.  
Added floating window mode (double-click the red X in floating window mode to close the window).  
Added system tray.  
Optimized some small logic.

5.  
Added AI reference image generation, but the effect is not ideal currently.  
Requires the API key provided by chatanyway.com.  
Free API keys are available on GitHub, allowing 200 requests per day (AI image generation requires a paid key).

6.  
Enhanced the screenshot feature to select screenshot areas pixel by pixel.  
Function keys: Up, Down, Left, Right.  
Ctrl + Up, Down, Left, Right.  
Space + Up, Down, Left, Right.
